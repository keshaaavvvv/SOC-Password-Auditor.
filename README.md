🛡️ SOC Sentinel: Advanced Password Security AuditorSOC Sentinel is a modern, GUI-based security tool designed for Security Operations Center (SOC) environments. It moves beyond simple character checks to provide a deep technical analysis of credential resilience using mathematical entropy and brute-force modeling.📸 Interface OverviewStandard StateAudit in ProgressClean, terminal-inspired interfaceReal-time entropy & risk analysis

✨ Key FeaturesShannon Entropy Analysis: Calculates the theoretical randomness of a password in bits.Brute-Force Modeling: Estimates "Time-to-Crack" based on high-performance GPU cluster speeds (10 billion guesses/sec).Security Policy Auditing: Flags specific risks like casing diversity, numerical requirements, and length vulnerabilities.NIST 800-63B Guidelines: Built with a focus on length and complexity over simple pattern replacement.Cryptographically Secure Generator: Uses Python's secrets module for high-entropy passphrase generation.🚀 Technical Deep-DiveAs an aspiring SOC L1 Analyst, I designed this tool to focus on Risk Assessment:Entropy Formula: Uses $E = \log_2(R^L)$ to quantify password strength.Modern UI: Developed with CustomTkinter for a professional, dark-mode dashboard experience.Audit Logs: Features a "Threat Analysis" console that provides live feedback on security compliance.🛠️ Installation & UsageClone the repository:Bashgit clone https://github.com/keshaaavvvv/SOC-Password-Auditor.git
